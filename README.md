
# Portfolio Website (React + Vite)

This is a modern, resume-style portfolio website built with React.js, Vite, HTML, CSS, and JavaScript.

## Sections
- **Home**
- **About**
- **Skills**
- **Projects**
- **Contact**

The design is clean and professional, ideal for showcasing your experience, skills, and projects.

## Getting Started

To start the development server:

```sh
npm run dev
```

To build for production:

```sh
npm run build
```

---
This project was bootstrapped with [Vite](https://vitejs.dev/) and [React](https://react.dev/).
